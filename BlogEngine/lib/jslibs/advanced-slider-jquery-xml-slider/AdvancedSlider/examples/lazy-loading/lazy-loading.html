Highlighted Features:

  • Lazy Image Loading. The images will not load when the page is opened but only when you navigate to the slide. Also, you can choose to preload the nearby images.
  • Multiple Navigation Modes. You can navigate throught the slides using the slider's controls but also using the keyboard (left and right arrow buttons). The thumbnail scroller also supports mouse scroll and mouse wheel navigation.
